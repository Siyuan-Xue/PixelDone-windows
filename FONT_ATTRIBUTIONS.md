# Bundled Font Attributions

PixelDone bundles the following unmodified variable font files from the
[Google Fonts repository](https://github.com/google/fonts):

- Source Sans 3
- Source Serif 4
- Noto Sans SC
- Noto Serif SC
- Noto Sans Arabic
- Noto Naskh Arabic

Each family is distributed under the SIL Open Font License, Version 1.1.
The complete license text is stored in `licenses/fonts/OFL-1.1.txt`.
Copyright and reserved-font-name notices remain embedded in the original
font binaries.
